# AIPM — Cohort 2

Everything you need for the program lives in this repo. Materials land here session by session — right now you have Sessions 1–2.

## Start here

1. **Before Session 1:** read [pre-read_ai-primer.md](pre-read_ai-primer.md).
2. **In Session 1:** work from [student-handbook-session-1.md](student-handbook-session-1.md) — it's the spine for the whole session.
3. **After Session 1:** ship [build-lab-1.md](build-lab-1.md) (the resume screener) and pick your capstone direction from [capstone-problems.md](capstone-problems.md).
4. **In Session 2:** work from [student-handbook-session-2.md](student-handbook-session-2.md).
5. **After Session 2:** ship [build-lab-2.md](build-lab-2.md) (the interview agent).

## What's in here

| File | What it is |
|---|---|
| `pre-read_ai-primer.md` | Pre-read — how LLMs actually work, in PM terms |
| `student-handbook-session-1.md` | Session 1 handbook — the working session, step by step |
| `student-handbook-session-2.md` | Session 2 handbook — agent architectures, root cause, HITL |
| `case-study-talentscope.md` | The TalentScope case study we build on all program |
| `build-lab-1.md` | Build Lab 1 — ship a resume screener |
| `build-lab-2.md` | Build Lab 2 — build the interview agent |
| `bonus-labs.md` | Optional labs A & B (PM Skills, PM Brain) |
| `capstone-problems.md` | Capstone directions — pick one, teams of 2–3 |
| `talentscope-data/` | Test data for Sessions 1–2 and Build Labs 1–2 |

Your own outputs (opportunity memo, product brief, PRD section, lab builds) go in a `my_work/` folder you create locally — not committed here.

Sessions 3–4 materials get added before each session.
